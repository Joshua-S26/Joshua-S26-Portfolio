# Joshua Short: Portfolio

Joshua Short is a Power Platform and Power BI developer who builds dashboards and semantic models that leaders run the business on, along with the automations and low-code apps that feed them. He is based in Sacramento, California, and works remotely across the US.

**Certifications:** PL-300, DP-600, DP-700, PL-500, PL-900, DP-900, AWS Cloud Practitioner

## Projects

- **Invoice exceptions: a full-stack Power Platform design.** A solution sketch for a finance exceptions workflow: a canvas app, an embedded Power BI hub, a Teams approval flow, and a Copilot Studio agent, all on one Dataverse model.
  [View the project](https://joshua-s26.github.io/Joshua-S26-Portfolio/projects/invoice-exceptions.html)

- **O-list delivery insights: Power BI report as code.** A Power BI dashboard and case study on the Olist Brazilian e-commerce dataset, with the semantic model and report authored as code in TMDL and PBIR.
  [View the project](https://joshua-s26.github.io/Joshua-S26-Portfolio/projects/olist-delivery.html)

## How this repo is organized

The site files (`index.html`, `projects/`, `assets/`) sit at the root of the repository, so GitHub Pages can serve them directly from `main`. The Power BI project, the TMDL semantic model, and the PBIR report live under `report/`, in the `OlistDeliveryInsights` project.
